## 一、克隆/新建项目步骤概要
1. 打开剪映 PC 端 / 移动端
2. 新建项目（横屏/竖屏均可）
3. 添加素材（可选：背景图/空白视频）
4. 进入「文本」→「文本朗读」
5. 选择音色、输入朗读文本
6. 生成配音轨道
7. 调整音量、时长后导出

## 二、导出文件格式
- 配音音频导出格式：MP3
- 音频采样率：44100 Hz
- 声道：单声道 / 双声道
- 文件名：dubbing.mp3

## 三、配音文件位置
本地路径：test_audio/dubbing.mp3

## 四、文件校验（可选）
文件大小：约 1.2MB
MD5：可自行生成后填写
